# LangChain Introducci√≥n con HuggingFace LLMs y PDF QA

Este repositorio contiene un ejemplo de uso de **LangChain** para:

1. Cargar un modelo LLM de HuggingFace.
2. Crear un prompt y generar respuestas.
3. Cargar documentos PDF online.
4. Dividir el contenido en chunks.
5. Crear embeddings de texto.
6. Almacenar documentos en un vectorstore (Chroma).
7. Realizar QA (Preguntas-Respuestas) con `ConversationalRetrievalChain`.
8. Usar herramientas externas (ej: Arxiv) con un agente tipo `ZERO_SHOT_REACT_DESCRIPTION`.
